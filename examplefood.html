Regression Analysis:

The correlation between the dependent and explanatory variables is implied by the multiple R. For instance, if R-1, it indicates that the dependent and independent variables have a perfect positive relationship. Once more, if R=0, it indicates that there is no correlation between the predicted and observed numbers. Here, the value of R is 0.0632, indicating that the independent and dependent variables have a somewhat linear connection.

The square value of multiple R in the regression model is known as the R-square, or coefficient of determination. It signifies the proportion of the dependent variable's variance that can be accounted for by the independent variables. R-square values between 0.3 and 0.5 suggest a moderate effect size, values above 0.7 indicate a strong impact size, and values below 0.3 indicate a poor effect size. Here, the R-square value is 0.0766, indicating that the independent variables account for 7.66%   of the variation in the non-performing loan ratio.

The number of terms in a multiple regression model is taken into account by adjusted R-square.
